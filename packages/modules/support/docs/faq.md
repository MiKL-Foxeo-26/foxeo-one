# Module Support — FAQ

## Questions fréquentes

### Comment signaler un problème ?
Cliquez sur "Signaler" dans le menu utilisateur, remplissez le formulaire et soumettez.

### Quels types de signalement sont disponibles ?
- **Bug** : Problème technique ou dysfonctionnement
- **Question** : Demande d'information ou clarification
- **Suggestion** : Idée d'amélioration

### Comment suivre mon signalement ?
Accédez à la page "Support" dans votre dashboard pour voir tous vos tickets et leur statut.

### Quels sont les statuts possibles ?
- **Ouvert** : Ticket créé, en attente de traitement
- **En cours** : MiKL travaille dessus
- **Résolu** : Problème corrigé
- **Fermé** : Ticket archivé

### Puis-je joindre une capture d'écran ?
Oui, le formulaire de signalement permet de joindre une image (PNG, JPG, WebP).
