## Bibliothèques de Composants Open Source

### Uiverse.io

| Attribut | Détail |
|----------|--------|
| **URL** | https://uiverse.io/ |
| **GitHub** | https://github.com/uiverse-io/galaxy |
| **Licence** | MIT (libre usage personnel et commercial) |
| **Éléments** | 7 167+ composants |
| **Communauté** | 282 470+ contributeurs |

**Technologies supportées :**
- HTML/CSS pur
- Tailwind CSS
- React (export)
- Figma (copier-coller direct)

**Catégories disponibles :**
- Boutons (`button`)
- Cartes (`card`)
- Loaders / Spinners (`loader`)
- Champs de saisie (`input`)
- Formulaires (`form`)
- Interrupteurs (`switch`, `toggle`)
- Cases à cocher (`checkbox`, `radio`)
- Info-bulles (`tooltip`)
- Effets animés (`animation`, `hover effect`)

**Styles disponibles :**
`minimal`, `modern`, `dark`, `gradient`, `neumorphism`, `3d`, `glow`, `minimalist`

**Quand utiliser :**
- Lors de la création du Design System
- Pendant l'implémentation des composants UI
- Pour accélérer le développement frontend

**Liens utiles :**
- Tous les éléments : https://uiverse.io/elements
- Recherche par tags : https://uiverse.io/tags/[tag-name]
- Discord communauté : https://discord.gg/KD8ba2uUpT
