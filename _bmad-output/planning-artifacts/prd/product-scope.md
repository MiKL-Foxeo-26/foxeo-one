# Product Scope

## MVP (P1) — Cas Client Référence

Le MVP est défini par le premier cas client réel : une association avec une assistante de direction qui gère tout (admin, formations, événements, adhésions).

**Objectif :** Démontrer tout le savoir-faire Foxeo sur un cas complet.

| Module | Contenu MVP | Priorité |
|--------|-------------|----------|
| **Site Public** | Vitrine + Espace adhérent + Inscriptions Stripe | P1 |
| **CRM** | Fiche adhérent unique, historique complet | P1 |
| **Formations** | Catalogue, inscriptions, paiements, Qualiopi | P1 |
| **Événements** | Création, publication, inscriptions, facturation | P1 |
| **Adhésions** | Campagne, renouvellement, suivi | P1 (déploiement Octobre) |
| **Élio One** | Assistant agentique (requêtes, actions, génération) | P1 |
| **Dashboard** | Vue globale, alertes, KPIs | P1 |

## Growth (P2) — Version 2

Analytics avancés, notifications push, PWA, compteur de valeur, 2FA, automatisations avancées.

## Vision (P3) — Futur

Multi-utilisateurs, API publique, gamification, multi-langue, marketplace de modules.
